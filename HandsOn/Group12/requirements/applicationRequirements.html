Proposed Application: AirViz Madrid

General Description

AirViz Madrid is a web application that allows users to visualize, analyze, and compare air quality across different areas of Madrid using linked open data.

Objectives

  • Display an interactive map of monitoring stations.
  • Visualize time series of pollutant levels.
  • Provide alerts when recommended thresholds are exceeded.
  • Enable historical and comparative analysis.

Functional Requirements

  • FR1: Display map with pollutant levels by station.
  • FR2: Filter by pollutant, date range, and location.
  • FR3: Generate interactive trend charts.
  • FR4: Allow CSV data export.

Non-functional Requirements

  • NFR1: Responsive and accessible interface.
  • NFR2: Automatic data updates.
  • NFR3: Compliance with RDF and Linked Data standards.
