Requirement check (R1–R6)

Each dataset is evaluated against the assignment requirements. R1–R4 are mandatory; R5–R6 are optional but discussed.

Portal: Open Data BCN — monthly CSV resource (e.g., 2024_11_Novembre_qualitat_aire_BCN.csv). License: CC BY 4.0. See the dataset and resource pages in the portal.

ReqSatisfies?Justification
R1 — Smart city domainYesEnvironment / urban air quality.
R2 — Data available as CSVYesThe portal publishes monthly CSV files with station readings.
R3 — Open licenseYesCreative Commons BY 4.0 stated in the dataset page.
R4 — Easily linkable to generic entitiesYesIncludes station IDs and coordinates; linkable to schema:Place, geo:Point, districts, etc.
R5 — Documentation existsYesThe dataset page describes fields and resources.
